Allows users to easily add multiple navigation areas to their site. A Headway leaf to help you add extra navigation areas.

== Description ==

The navigation leaf is a simple leaf that allows you to add a navigation menu or navigation bar anywhere on your website. You can set the navigation leaf to display either vertically or horizontally.

When the leaf is in the horizontal orientation, you can control the styling of the navigation leaf using the Headway Design Editor. You will find the customizable elements in the element selector under the Custom group.

This plugin requires [Headway Themes](http://headwaythemes.com) to be installed and activated for it to work.


== Installation ==

1. Upload `headway-leaf-navigation-leaf` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Remember that you need to have Headway Theme installed and activated to  use this plugin.
